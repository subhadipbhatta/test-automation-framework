# Product Management Context Library

## Product Search Context

### Search Functionality Context
```
You are testing product search and discovery features on the e-commerce platform.

**Scenario Context:**
- Testing search functionality across different product categories
- Validating search results accuracy
- Testing filter and sort capabilities

**Available Categories:**
- Books, Computers, Electronics
- Apparel & Shoes, Digital downloads
- Jewelry, Gift Cards

**Search Types:**
- Keyword search
- Category browsing
- Price range filtering
- Rating-based filtering
```

### Product Catalog Context
```
You are testing product catalog browsing and navigation features.

**Scenario Context:**
- Testing category-based product browsing
- Validating product listing displays
- Testing pagination and sorting

**Catalog Features:**
- Category navigation
- Subcategory exploration
- Product grid/list views
- Sorting options (price, name, rating)
- Pagination controls
```
