# Shopping Cart Context Library

## Cart Management Context

### Cart Operations Context
```
You are testing shopping cart functionality including add, remove, and update operations.

**Scenario Context:**
- Testing add to cart functionality
- Validating cart item management
- Testing quantity updates
- Verifying cart persistence

**Test Scenarios:**
- Add single item to cart
- Add multiple items
- Update quantities
- Remove items
- Clear entire cart
- Cart persistence across sessions
```

### Cart Validation Context
```
You are testing cart data accuracy and calculations.

**Scenario Context:**
- Validating cart item information
- Testing price calculations
- Verifying cart totals
- Testing discount applications

**Validation Points:**
- Item details accuracy
- Quantity calculations
- Price calculations
- Tax calculations (if applicable)
- Shipping calculations
- Total amount accuracy
```

## Cart Persistence Context

### Session Cart Context
```
You are testing cart persistence across user sessions and page navigation.

**Scenario Context:**
- Testing cart state maintenance
- Validating cart persistence for logged-in users
- Testing guest cart behavior

**Persistence Scenarios:**
- Cart maintenance during site navigation
- Cart persistence after login/logout
- Guest cart to registered user cart transfer
- Cart recovery after session timeout
```

## Cart User Experience Context

### Cart Interface Context
```
You are testing the shopping cart user interface and experience.

**Scenario Context:**
- Testing cart page layout and functionality
- Validating cart notifications and updates
- Testing cart accessibility features

**UI Elements:**
- Cart page design and layout
- Cart counter/badge updates
- Add to cart notifications
- Cart page responsiveness
- Cart editing controls
```
