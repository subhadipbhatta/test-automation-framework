# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02-12

### Added

- Initial framework release
- Page Object Model (POM) implementation
- Web automation with Playwright
- API automation with aiohttp
- Salesforce OAuth2 authentication
- Salesforce REST API client
- Salesforce UI automation
- MCP server for Playwright integration
- Configuration management system
- Test data generation utilities
- Wait utilities with conditional polling
- Pytest fixtures and configuration
- Example tests for web, API, and Salesforce
- Comprehensive documentation
- Logging configuration
- Screenshot on failure capability
- Assertion utilities
- Support for environment-based configuration

### Features

- Async/await support throughout
- Cross-browser testing (Chromium, Firefox, WebKit)
- Multiple authentication methods
- SOQL query support
- Dynamic wait mechanisms
- Test reporting with HTML and Allure
- Custom markers for test categorization

### Documentation

- README with full setup instructions
- Architecture guide
- API testing guide
- Salesforce automation guide
- Quick start guide
- Contributing guidelines
